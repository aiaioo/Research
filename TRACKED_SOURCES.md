# AI Research Paper Curation Sources

Manually or community-curated sites for discovering high-quality AI papers. See the final section for algorithmic/uncurated feeds.

---

## Daily

| Site | URL | Notes |
|------|-----|-------|
| Hugging Face Papers | https://huggingface.co/papers | Community upvotes papers each day; typically 10–30 surfaced |
| Scholar Inbox Trending | https://www.scholar-inbox.com/trending | Personalized + trending view updated daily |
| Emergent Mind | https://www.emergentmind.com | Highlights arxiv papers getting social traction; good signal-to-noise |
| AlphaSignal | https://alphasignal.ai | Daily digest of top ML papers and repos; brief summaries |
| TLDR AI (newsletter) | https://tldr.tech/ai | Daily email; 3–5 papers/articles chosen by editors |

## Social feeds worth following for daily picks

These aren't standalone sites but reliably surface top papers first:

- **@_akhaliq** (Abubakar Abid) on X/Twitter — posts new papers daily, often before they trend elsewhere
- **r/MachineLearning** on Reddit — community-voted; quality varies but top posts are usually solid
- **Papers With Code newsletter** — https://paperswithcode.com (subscribe for weekly digest)

## Algorithmic / Uncurated Feeds

High-volume or automated sources excluded from the curated sections above. Useful for breadth but require more filtering.

| Site | URL | Update Cadence | Notes |
|------|-----|----------------|-------|
| arXiv cs.AI / cs.LG / stat.ML | https://arxiv.org/list/cs.LG/recent | Daily | Raw firehose; hundreds of papers per day |
| arXiv Sanity Lite | https://arxiv-sanity-lite.com | Daily | SVD-based recommendations; useful once seeded with preferences |
| Papers With Code (main feed) | https://paperswithcode.com/latest | Daily | Algorithmically ranks by GitHub stars + citations; broad but noisy |
| Semantic Scholar | https://www.semanticscholar.org | Continuous | Large corpus search; no editorial curation but good for discovery by topic |
| Connected Papers | https://www.connectedpapers.com | On-demand | Graph of related papers around a seed paper; exploratory rather than feed-based |
| Google Scholar Alerts | https://scholar.google.com/scholar_alerts | As indexed | Email alerts for keywords or authors; quality depends entirely on query design |
| ResearchGate | https://www.researchgate.net | Continuous | Social network for researchers; surfaces followed authors' new papers |
